# Contributing

Thank you for considering a contribution to LightWorship.

## Useful Areas

- Windows 7/Windows 10 compatibility testing
- Projection-window reliability
- Bible reference parsing
- Local/offline transcription adapters
- Licensed Bible/provider import scaffolding
- Accessibility and keyboard workflow improvements
- Installer and portable-build testing

## Development

1. Open `LightWorship.sln` in Visual Studio or build with MSBuild.
2. Keep generated binaries, installers, and local provider tools out of Git.
3. Put local API keys in `data/settings.json`; never commit them.
4. Run the automated tests listed in `docs/TESTING.md`.

## Pull Requests

Please include:

- What changed
- How it was tested
- Any compatibility notes for older Windows machines
- Screenshots or short clips for UI/projection changes when helpful
