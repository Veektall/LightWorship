# Code of Conduct

LightWorship is intended for churches, ministries, and community teams. Contributions should be respectful, practical, and focused on helping operators run services reliably.

Please avoid harassment, personal attacks, discriminatory language, or sharing private worship-team, church, or attendee information.

Maintainers may remove comments, issues, or contributions that are abusive, unsafe, spammy, or unrelated to the project.
